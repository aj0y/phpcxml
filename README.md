PHP-CXML
==========================
A PHP library to deal with cXML.

- PSR compatible


Requirements
------------
- PHP >= 5.6


Install
-------

### Composer

Add a dependency on `beethoven/phpcxml` to your project's `composer.json` file.

```javascript
{
    "require": {
        "beethoven/phpcxml": "0.0.1-alpha"
    }
}
```
